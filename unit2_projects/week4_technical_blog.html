    Looping

    Loops can become very powerful tools in your kit as long as you understand some general concepts involved with selecting the proper loop for your problem. The following list will contain examples of the basics loops provided in Ruby.

    • While - while condition { block }
    • Until - until condition { block }
    • For - for variable in expression { block }
    • Each - each {|index| block}

    You'll want to employ a "while loop" if you don't know how many times you will need to do something but you can track a condition that is changing. An "until loop" can be used when you know what the final condition will be. A "for loop" is to be used when you know exactly how many times you'll need to execute a particular block. Finally, an "each loop" is to be used when you know the range and would like to apply a certain block to each element of that range.

    It seems the more I learn about programming, the more it leads me back to the fundamental workings of basic loops and methods. All we are doing is creating tasks with methods and having the computer loop through those defined methods over and over again.
