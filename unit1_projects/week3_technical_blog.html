Week 3 - JavaScript Favoritism

Why is JavaScript so popular in Web Development?

JavaScript is a lightweight, dynamic, and easy to use language. That ease of use is the leading reason many businesses are turning to JS to power their applications. In today's world, the market is constantly changing. In order to remain competitive one must have a product that can be altered to take advantage of the most current trends. JavaScript is the very language to facilitate this adapative nature.

With the recent popularity of Cloud Computing, JavaScript has seen a bit more server-side esposure. When implemented using the Node.js platform (based on Google's V8 JavaScript engine), JS can be used as an easily adaptable backend language. The popularity of this trend continues to grow as more and more established names hop on the bandwagon.

We should not overlook JavaScript being a Object Oriented Programming language. This is the reason for this language being so dynamic in our deployments. Our ability to construct objects and then manipulate them allows for much more flexibility when adding content to our sites.
